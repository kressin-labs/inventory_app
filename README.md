# Inventory Management Application (Frontend)

This is the client-side application for the inventory management system, built using React, TypeScript, and Vite.

## Key Features

* **Product Card View:** Products are displayed in a clean, responsive card grid.
* **User Authentication:** Login via a slide-in modal (handles Admin and standard User roles).
* **Shopping Cart:** Users can add multiple items to a persistent cart and purchase all items via the API.
* **Admin Controls:** Admin users can create, modify, and delete products directly from the shop page.
* **Visual Feedback:** Uses toast notifications and card highlighting for immediate feedback on cart actions.

## Technology Stack

* **Framework:** React
* **Language:** TypeScript
* **Build Tool:** Vite

## Local Development Setup

### Prerequisites

* Node.js (LTS version)
* Access to the running **Inventory Backend API** (Java Spring Boot).
